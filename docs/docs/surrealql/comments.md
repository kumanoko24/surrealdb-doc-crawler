# Comments


In SurrealQL, comments can be written in a number of different ways.

```
/*In SurrealQL, comments can be written as single-lineor multi-line comments, and comments can be used andinterspersed within statements.*/SELECT * FROM /* get all users */ user;# There are a number of ways to use single-line commentsSELECT * FROM user;// Alternatively using two forward-slash charactersSELECT * FROM user;-- Another way is to use two dash charactersSELECT * FROM user;
```
